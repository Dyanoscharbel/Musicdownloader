🎵 Spotify Downloader Pro

Téléchargez vos playlists Spotify en audio ou vidéo, en toute simplicité !

🚀 Qu'est-ce que c'est ?

Spotify Downloader Pro est une application de bureau qui vous permet de :

  • ✅ Importer vos playlists Spotify
  • ✅ Télécharger les titres via YouTube
  • ✅ Choisir le format : audio (M4A) ou vidéo (MP4)
  • ✅ Profiter d'une interface moderne, intuitive et efficace

🌟 Fonctionnalités principales

  • 🎧 Liberté totale : Téléchargez vos musiques préférées, écoutez-les hors ligne
  • 🖼️ Gestion des playlists : Visualisez les pochettes, sélectionnez les titres
  • 🧭 Navigation fluide : Interface en onglets (Playlists, Téléchargements, Guide...)
  • 🛠️ Configuration rapide : Ajout simple des clés API Spotify
  • 📁 Téléchargements intelligents : Pas de doublons, barre de progression, journal détaillé
  • 🎬 Support FFmpeg : Conversion audio/vidéo automatique
  • 🔐 Système sécurisé : Connexion/inscription avec gestion de compte
